# Project Rules — Always Follow

## Tech Stack (non-negotiable)
- Framework: Next.js 14+ with App Router
- Styling: Tailwind CSS only — no inline styles, no CSS modules
- Animations: Framer Motion only — no GSAP, no CSS keyframes
- Icons: HugeIcons only — https://hugeicons.com — no Lucide, no Heroicons
- Language: TypeScript only — no plain JavaScript files

## Design References (browse before designing)
Before proposing or building any UI, always visit:
- Landing pages → https://landing.love
- SaaS patterns → https://saaspo.com
- Navbar patterns → https://navbar.gallery
- Animation ideas → https://appmotion.design
- Component patterns → https://component.gallery

## Code Rules
- Every component must be reusable and isolated
- No hardcoded colors — use design tokens only
- Mobile-first always
- No placeholder content — generate realistic copy

## Behavior Rules
- Never modify existing code without showing a plan first
- Always ask for approval before implementing
- If a better technology exists, mention it but still use the stack above
